# 📌 Project Overview

Micro-Task and Earning Platform is a role-based system that connects Workers, Buyers, and Admins.

- 👷 Workers complete tasks and earn coins  
- 🧑‍💼 Buyers create tasks and manage workers  
- 🛡️ Admin controls users, tasks, withdrawals, and platform security  

---

# ✨ Key Features

## 🔐 Authentication System
- Email/password authentication
- Google login support
- JWT-based authentication
- Role-based access (Worker / Buyer / Admin)
- Default coin system:
  - Worker: 100 coins
  - Buyer: 200 coins
